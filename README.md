# Kharch Kaabu - Finance Dashboard UI

A simple finance dashboard to track income, expenses, and spending patterns.

Built as part of a frontend assignment using HTML, CSS, and JavaScript.

## Features
- Financial summary (balance, income, expenses)
- Transaction list with basic filtering/search
- Charts for trends and category breakdown
- Role-based UI (Admin / Viewer)
- Basic insights

## Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js

## How to Run
Open `index.html` in a browser.

## Note
This project fulfills the required frontend assignment features.
